# CitationProverbe

Permet à A.V.A.T.A.R. de diffuser des citations directement à la voix. Aléatoire

- This plugin is an add-on for the [A.V.A.T.A.R](https://avatar-home-automation.github.io/docs) framework.

## 🎯 Usage
Commandes :
- raconte une citation/proverbe,
- donne-moi une citation/proverbe,
- une autre citation/un autre proverbe.

## Multi-room

The `CitationProverbe` plugin is fully multi-room.

## Multi-language

The `CitationProverbe` plugin relies solely on the system's available languages.
